# AngularBase

Basic project used for quickly cloning to test things out

```
git clone https://github.com/IsaacSomething/angular-base.git
```

```
cd angular-base/ && npm install
```

- "@angular/core": "^14.2.7"
- "@angular/material": "^14.2.5"
